As part of my final project through Portland Code School, I participated in a group project. The results of these questions return video results of exercise and fitness routines. I was mainly responsible for the YouTube API request and node server work. I also handled all Orchestrate data pushes and calls through the node server with some additional backbone view and model manipulation.
